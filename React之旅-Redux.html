  1. createStore // 创建store
    createStore(reducer,window.REDUX_DEVTOOLS_EXTENSION && window.REDUX_DEVTOOLS_EXTENSION());
  2. store.dispatch(action) // 修改Store中属性的唯一方法
  3. store.getState() // 返回Store中所有的属性值
  4. store.subscribe() // 监听Store中有属性值发生改变 ；立即调用用户设置的方法
  5. combineReducers({}), // 合并多个小的reducer 返回一个大的reducer 类似vuex模块的概念

yarn 常用的命令

  1. yarn init 初始化
  2. yarn add * 安装一个库
  3. yarn remove 删除一个库
  4. yarn install 安装一个框架所有的包
  5. yarn upgrade * 升级一个包

    注意

  6. reducer必须是纯函数
  7. 什么是纯函数：给出固定的输入，必须给出固定的输出，而且不会有副作用
  8. 只有store才能修改store中的值 redux中修改即store中修改
  9. redux的属性是单一数据流
  10. redux中不能又异步操错，setTimout,setInterval 因为他是纯函数
